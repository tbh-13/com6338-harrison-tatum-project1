This is a searchable index website for the trading card game, Magic the Gathering.

Users can input the name of a card, or a word that appears within the card name, and the results will pull from the https://api.magicthegathering.io API to show all cards that match the inputted value. 

The site will render the card img (if there is no img, the back of the card will show), card name, color identity (WBBRG), type, rarity, and set the card is in.
